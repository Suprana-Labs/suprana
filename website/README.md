# Suprana Website

Official website for the Suprana blockchain.

## About

This is the landing page and documentation portal for Suprana - a high-performance blockchain with 10 billion SUP native tokens.

## Features

- **Homepage**: Overview of Suprana blockchain
- **Whitepaper**: Technical documentation and specifications
- **Network**: Testnet and mainnet information
- **Ecosystem**: Tools, wallet, and explorer information
- **Tools**: Developer tools and CLI references
- **Documentation Links**: Direct links to GitHub docs

## Tech Stack

- **Framework**: Next.js 14
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Deployment**: Static export

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run start
```

The site will be available at `http://localhost:3000`

## Deployment

This site is configured for static export and can be deployed to:
- GitHub Pages
- Vercel
- Netlify
- Any static hosting service

```bash
# Build static site
npm run build

# Output will be in the `out/` directory
```

## Pages

- `/` - Homepage
- `/whitepaper` - Suprana whitepaper
- `/network` - Network information (testnet/mainnet)
- `/ecosystem` - Ecosystem tools and applications
- `/tools` - Developer tools and resources

## Contributing

To update the website:

1. Make changes to the source files in `src/`
2. Test locally with `npm run dev`
3. Build with `npm run build`
4. Commit and push to repository

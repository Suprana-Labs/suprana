## üìñ About Suprana

Suprana is a high-performance blockchain featuring:
- **Native Token**: SUP (10 billion total supply)
- **Block Time**: ~400ms
- **Consensus**: Proof of History (PoH) + Proof of Stake (PoS)
- **Decimals**: 9 (1 SUP = 1,000,000,000 lamports)

- **Disk**: 500GB SSD (for full validator nodes)
